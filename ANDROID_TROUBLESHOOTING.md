Android prefers to use cellular data even when WiFi is available.

This application attempts to use the WiFi network by default but
that will only work if you connect your device to the the WiFi
network first. So if the device you want to send the GPS 
coordinates to is on a local WiFi network connect to it first 
before opening the application.

If you launched the app before you connected to the WiFi network
you will need to close the app and launch it again.
